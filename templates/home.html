About Project

AI Image Classification using CNN

This project uses Deep Learning and Computer Vision techniques to classify uploaded images as either Cat 🐱 or Dog 🐶. The model is built using TensorFlow and Keras with a Convolutional Neural Network (CNN) architecture trained on a custom dataset containing 1000+ images. Users can upload images through an interactive Flask web application and receive real-time predictions with confidence scores.
